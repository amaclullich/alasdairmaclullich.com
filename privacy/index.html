Privacy

Privacy on this website

This is a static website with no forms, accounts, comments or newsletters. Optional audience measurement is used only if you allow it.

Information collected

Information you submit

There is no form on this website, so it does not directly collect names, email addresses, messages or other information from visitors.

Technical records

The site is hosted using GitHub Pages. As with most web hosting, GitHub may process technical information such as IP addresses and browser details for delivery, security and operational purposes. GitHub describes its handling of this information in the GitHub General Privacy Statement.

Cookies and analytics

Google Analytics is available as an optional audience-measurement service. It is not loaded unless you select Allow analytics. If you accept, it can record information such as page views, approximate location, device and browser details, scrolling and outbound-link clicks. Google Analytics may then set identifiers such as _ga. Analytics is not used for advertising on this website.

If you decline, the Google Analytics tag is not loaded. Your choice is stored in your browser's local storage so that the site can remember it. You can review or change that choice at any time.

Google explains how it safeguards data in Safeguarding your data and provides further information in its Privacy Policy.

Other websites

This website links to external sites, including university, research, professional and public-resource websites. Those services have their own privacy practices. Following a link takes you to a separate website whose operator is responsible for it.
